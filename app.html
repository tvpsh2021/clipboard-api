clipboardData API : What's in clipboard?

  • Just paste anything to whole html body area.
  • Test OS: macOS, Windows.
  • Test Browser: Chrome latest.
  • Every single action of copy(ctrl + c / cmd + c) contains ONLY one object.
    • clipboardData -> DataTransfer
    • clipboardData.files -> FileList
    • clipboardData.items -> DataTransferItemList
    • clipboardData.types -> Array(2)
  • (macOS) Once you copy many files, clipboardData API will present the last file you've selected.
  • (macOS) .pdf file will be identify as a image file.
  • Depend on what you've copied, items object will be different:
    condition macOS items[0].kind macOS items[1].kind Windows items[0].kind Windows items[1].kind
    Pure string string undefined string undefined
    String with hidden metadata (e.g. html) string string string string
    File(s) string file undefined undefined
    Print screen to clipboard file undefined file undefined
  • You can see more detail in W3C Document.
